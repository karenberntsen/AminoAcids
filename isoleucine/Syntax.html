Syntax

This section describes the syntax used throughout the webpage.
= Equal sign. Value must be equal to the term defined after the '=' sign.
[] These signs represent a closed interval. [a,b] means that all values lay between a and b, including a and b itself.
<> These signs represent an open interval. <a,b> means that all values lay between a and b, excluding a and b itself.
Closed and open intervals can be combined such as [a,b> or <a,b]. Often though, the border signs []<> are somewhat meaningless, because the chance of finding an angle of exactly 120.000000000 degrees is rather small.
The secondary structure is divided in 'H', 'S' and 'C'. 'H' indicates a pure α-helix or a 3/10-helix (excluding π-helix), 'S' indicates a fully hydrogenbonded β-strand residue which DSSP calls 'E'. 'C' indicates everything that is not any form of helix or the strand called 'E' by DSSP.
WHAT IF uses a composit file quality cutoff comprising a dozen selection criteria (DACA-score better than -2; water molecules must be present; only a few residues allowed with missing side-chain; protein atoms must make up > 75 % of the file (not counting waters); etc.). See WHAT CHECK documentation for details. Use of this criterium eliminates about 3% of all PDB-files solved at better than 3.0 Å resolution. The use of this criterium is indicated by 'use_file_db=TRUE'.
